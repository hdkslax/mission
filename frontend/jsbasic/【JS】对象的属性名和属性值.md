# 属性名
* 对象的属性名不强制要求遵守标识符的规范，但尽量按照标识符的规范
	```
	obj.var = "hello";
	```
* 若要使用特殊的属性名，不能使用.的方式来操作。要用 对象[“属性名”] = 属性值，读取也要用这种方式
	```
	obj.123 = 789; // 报错
	obj["123"] = 789; // 正确，但括号里只能是字符串
	```
* 运算符优先级中，`.`、`[]`、`new` 优先级最高，都是用来操作对象的。
* 使用 [ ] 这种形式去操作属性，更加灵活。在 [ ] 中可以直接传递一个变量，这样变量值是多少就会读取那个属性。
	```
	obj["123"] = 789;
	var n = "123";
	console.log(obj[n]);
	```
