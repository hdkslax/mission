# [C] printf

### printf中的对齐输出

###### 右对齐

数字12占3个数字宽，3占6个数字宽：

```c
printf("%3d %6d\n", 12, 3); //  12      3
```

### printf 格式化输出

```
%d		按照十进制整形数打印
%6d 	按照十进制整形数打印，至少6个字符宽
%f		按照浮点数打印 float double都用这个
%6f		按照浮点数打印，至少6个字符宽
%.2f	按照浮点数打印，小数点后有两位小数
%6.2	按照浮点数打印，至少6个字符宽，小数点后有两个小数
%o		八进制
%x		十六进制
%c		字符
%s		字符串
%%		百分号
```
