2. 漏洞利用之IPC$：

   ```bash
   net use  \\10.1.1.2\ipc$ 密码 /user:用户   #远程访问10.1.1.2上的ipc$
   net use f: \\10.1.1.2\share 密码 /user:用户 #将远程地址10.1.1.2上的share文件夹作为我的f盘
   net use f: /del #删除上一条命令中的f盘
   net use * /del #删除所有远程映射
   net use #查看所有远程映射
   ```
